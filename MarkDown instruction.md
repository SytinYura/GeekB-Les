# Инструкция по работе с Markdown syntax

Основные команды:

* git init - началот работы
* git add (название файла) - сохранить
* git status - проверить статус
* git commit фикация
* git log - проверить f
* git commit -m "текст комментария"
* git diff - чем отличаются версси
